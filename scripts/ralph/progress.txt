## Codebase Patterns
- CA1056 analyzer treats `string? PhotoUrl` as error on Domain entities ‚Äî suppress with `#pragma warning disable CA1056` around the property
- IDE1006 requires `_` prefix for private static readonly fields in Chairly.Api (e.g. `_jsonOptions`)
- MA0002 requires StringComparer on `new Dictionary<string, T>()` ‚Äî use `StringComparer.OrdinalIgnoreCase`
- CA2208/MA0015 require `ArgumentOutOfRangeException` to use a local parameter name, not `nameof(property.Subproperty)` ‚Äî extract to local variable first
- Slices in the same bounded context (e.g. Staff) can reference each other: UpdateStaffMemberHandler and Deactivate/ReactivateStaffMemberHandler call `CreateStaffMemberHandler.ToResponse()` and `ParseRole()` (declared `internal static`)
- dotnet format auto-fixes migration file encoding issues (CHARSET error) ‚Äî always run `dotnet format` before `dotnet format --verify-no-changes`
- `#pragma warning disable CA1812` is needed on all handler classes (instantiated via DI reflection)
