## Codebase Patterns
- API services use `inject(API_BASE_URL)` token from `@org/shared-lib` for base URL (not hardcoded `/api`)
- `simple-import-sort` ESLint plugin enforces export sort order — run `nx run chairly-lib:lint --fix` to auto-fix
- Models use `export type` (not `export`) in index.ts re-exports
- Staff API: deactivate/reactivate use PATCH with `null` body
- `[style.xxx]` property bindings require `allowBindToStyle: true` in ESLint `no-inline-styles` rule — already configured
